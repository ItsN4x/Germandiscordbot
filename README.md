# Germandiscordbot

System Befehle
give [userid]: Gibt einer bestimmten ID Vollzugriff auf alle Bot-Funktionen. (Nur für dich verfügbar).

revoke [userid]: Entzieht einer ID sofort alle Rechte am Bot. (Nur für dich verfügbar).

setup: Öffnet das grafische Menü zur Konfiguration des Bots.

status: Zeigt technische Daten, Ping-Zeiten und Server-Statistiken an.

Moderations Befehle
ban [user] [grund]: Entfernt ein Mitglied dauerhaft vom Server. Prüft automatisch die Rollen-Hierarchie.

kick [user] [grund]: Wirft ein Mitglied vom Server.

warn [user] [grund]: Registriert eine offizielle Verwarnung in der Datenbank des Bots.

timeout [user] [minuten]: Schaltet ein Mitglied für die angegebene Zeit stumm.

clear [anzahl]: Löscht die angegebene Menge an Nachrichten im Kanal (maximal 100).

lockdown [status]: Sperrt oder entsperrt die Schreibrechte für alle normalen Mitglieder im aktuellen Kanal.

Support und Server Befehle
ticket-panel: Erstellt eine Nachricht mit einem Button, über den Nutzer private Support-Kanäle öffnen können.

antilink [status]: Schaltet den automatischen Schutz gegen Internet-Links ein oder aus.

slowmode [sekunden]: Aktiviert eine Zeitverzögerung für das Schreiben von Nachrichten im Kanal.

userinfo [user]: Zeigt alle hinterlegten Daten, ID und Beitrittsdaten eines Nutzers an.

serverinfo: Gibt eine detaillierte Zusammenfassung über die technischen Daten des Discord-Servers aus.
